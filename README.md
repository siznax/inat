GitHub Pages app: <https://siznax.github.io/inat>

To install:

```shell
$ git clone https://github.com/siznax/inat.git
$ cd inat
$ yarn install
```

To run locally:

```shell
$ yarn start
```

To build and deploy:

```shell
$ yarn run deploy
```

Current state:

  * Initial API call populates app
  * Form allows composing new API URL
  * Form submit fetches new results
